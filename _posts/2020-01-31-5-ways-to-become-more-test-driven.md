1. ask from your clients if the business cases are needed
2. make sure that developers implement business cases correctly (BDD, Q2)
3. make sure that developers implement code in a way that they intended (TDD, Q1)
4. pipeline CI, automation etc.
5. when bugs occur, make sure that the root cause is identified and there is a test to reproduce it
